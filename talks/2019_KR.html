A few properties

Conjunction and disjunction are commutative, idempotent:

\[a \land b \equiv b \land a, a \lor b \equiv b \lor a.\] \[a \land a \equiv a, b \lor b \equiv b.\]

De Morgan laws:

\[a \land b \equiv \neg (\neg a \lor \neg b)\] \[a \lor b \equiv \neg (\neg a \land \neg b)\]

Negation is involutive: \(\neg \neg x \equiv x\).

Ontology

0th order logic

Given a set of propositional symbols \(a, b, c \in \mathcal{P}\):

  1. If \(a \in \mathcal{P}\), then \(a\) is a WFF.
  2. If \(f\) is a WFF, then \(\neg f\) is a WFF.
  3. If \(f_0, f_1\) are WFFs, then \(f_0 \ast f_1\) is a WFF.

Where \(\ast\) stands for a valid binary connectives (\(\land\), \(\lor\), \(\Rightarrow\), \(\Leftrightarrow\)).


E.g.: \(a \land (b \lor \neg \neg c)\) is a WFF.

\[BurningCoal \Rightarrow EarthsGettingHot.\] \[OnePlusTwoIsThree.\] \[\neg CatsAreBetterThanDogs.\] \[DoingFrontSquats \land BeingOutOfTheShape \Rightarrow Pain.\]

Probability theory is a zeroth-order logic

\[P(a) = 0 \equiv a \mbox { is false}.\] \[P(a) = 1 \equiv a \mbox { is true}.\]

1st order logic

Given a set of constants \(\mathcal{C}\), variables \(\mathcal{V}\) and functions \(\mathcal{F}\):

  1. If \(v \in \mathcal{V}\), then \(v\) is a term.
  2. If \(c \in \mathcal{C}\), then \(c\) is a term.
  3. If \(f \in \mathcal{F}\) and \(t_0, ...\) are terms, then \(f(t_0, ...)\) is a term.

Given a set of predicates \(P_0, P_1, P_2 \in \mathcal{P}\):

  1. If \(P \in \mathcal{P}\), then \(P\) is a WFF.
  2. If \(f\) is a WFF, then \(\neg f\) is a WFF.
  3. If \(f_0, f_1\) are WFFs, then \(f_0 \ast f_1\) is a WFF.
  4. If \(f\) is a WFF and \(v\) a variable, then \(\forall v: f\) and \(\exists v: f\) are a WFFs.

A predicate is a function from 0+ terms to a truth value

\[Awesome(Dog) > Awesome(Cat).\] \[\forall x \in \mathbb{N} \exists y \in \mathbb{N}: y = x + 1.\] \[\forall p, t: BackSquats(p, t) \land TotalBackSquats(p) = 0 \Rightarrow Pain(p, Thigh, t + 1).\]
\[\forall p: Smoking(p) \Rightarrow Cancer(p).\] \[\forall p_0, p_1, p_2: Friends(p_0, p_1) \land Friends(p_1, p_2) \Rightarrow Friends(p_0, p_2).\] \[\forall s, t \exists r: N(s, t + 1) = r \times N(s, t).\] \[\forall s: second law.\] \[0 \in \mathbb{N}.\] \[\forall n \in \mathbb{N}: n + 1 \in \mathbb{N}.\] \[\forall n \in \mathbb{N}: n > 1 \land n = \sum_{i \in [0, n)}i\times I(mod(n, i) = 0) \Rightarrow mod(n, 2) = 0.\]

Warning: First-order logic in A.I. mostly refer to a watered-down version without functions.

Higher-order logic

Advantages over first-order logic

More expressive: can quantify over higher-order structures.

Ubiquitus: almost all languages for mathematics are now based on type theory. It is used a foundation for both mathematics and programming languages.

Types! Their more sophisticated view of types make it easier to represent complex objects (vectors, matrices, networks, sum types).
