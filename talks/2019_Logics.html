The case for (and against) logic

Propositional logic (Zeroth-order logic)

Given a set of propositional symbols \(a, b, c \in \mathcal{P}\):

  1. If \(a \in \mathcal{P}\), then \(a\) is a WFF.
  2. If \(f\) is a WFF, then \(\neg f\) is a WFF.
  3. If \(f_0, f_1\) are WFFs, then \(f_0 \ast f_1\) is a WFF.

Where \(\ast\) stands for a valid binary connectives (\(\land\), \(\lor\), \(\Rightarrow\), \(\Leftrightarrow\)).

First-order logic

Given a set of predicates \(P_0, P_1, P_2 \in \mathcal{P}\):

  1. If \(P \in \mathcal{P}\), then \(P\) is a WFF.
  2. If \(f\) is a WFF, then \(\neg f\) is a WFF.
  3. If \(f_0, f_1\) are WFFs, then \(f_0 \ast f_1\) is a WFF.
  4. If \(f\) is a WFF and \(v\) a variable, then \(\forall v: f\) and \(\exists v: f\) are a WFFs.

A predicate is a function from 0+ terms to a truth value

Given a set of constants and variables, terms are:

  1. If \(v \in \mathcal{V}\), then \(v\) is a term.
  2. If \(c \in \mathcal{C}\), then \(c\) is a term.
  3. If \(f \in \mathcal{F}\) and \(t_0, t_1, t_2, t_2\) are terms.

Goals

Why C++?

  1. Highly optimized compilers (gcc, clang, intel)
  2. Modern C++ is actually nice (I swear)
  3. Flexible thanks to template metaprogramming
  4. Eigen, Ceres, Tensorflow, Apache Arrow, CUDA.
  5. Can easily write packages for Python (pybind11) and R (rcpp).

Concrete Advantages

  • Allows both logical and probabilistic reasoning.
  • Defines probability distribution over complex logical relationships.
  • Allows the introduction of mathematical and scientific knowledge.
  • Models are clear and safe.
  • Redefines data: programs, complex logical relationships, formulas are data.
